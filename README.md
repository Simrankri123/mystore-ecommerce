🛒 MyStore – E-commerce Web Application

📌 Overview

MyStore is a full-stack e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript. It provides a complete online shopping experience where users can browse products, add items to cart, and place orders.

🚀 Features

🔐 User Login System

🛍️ Product Listing with Categories

🔍 Live Search (AJAX)

🛒 Add to Cart with Quantity Update

💳 Checkout & Payment Flow

📦 Order History with Product Details

🎨 Modern UI & Responsive Design


🧠 Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL


⚙️ How to Run

Install XAMPP

Place the project folder inside htdocs

Start Apache and MySQL

Import the database in phpMyAdmin

Open in browser:
http://localhost/mystore
